# 🧶 Remo Crochet Works

> **Handcrafted with Love & Purpose**

A beautiful, fully responsive single-page website for **Remo Crochet Works** — a handmade crochet brand that creates keychains, flowers, bouquets, hairbands, handbags, bag charms, and home décor pieces.

🌐 **Live Website:** (https://remocrochetworks.clarisolvetech.com/)

---

## 🌸 About the Brand

*Remo stands for creating memories — our crochet pieces are not just products, they become part of someone's story.*

Remo Crochet Works is a handmade brand targeting women and girls who love cute, aesthetic, and meaningful handmade products. Every piece is crafted stitch by stitch with care, love, and purpose.

---

## ✨ Website Features

- **Fully Responsive** — works beautifully on mobile, tablet, and desktop
- **Single HTML File** — no frameworks, no build tools needed
- **Embedded Logo** — logo is base64-encoded directly into the HTML
- **Smooth Scrolling** — seamless navigation between sections
- **Scroll Animations** — elements reveal on scroll for a premium feel
- **Hover Effects** — subtle interactions throughout
- **Real Social Links** — Instagram, WhatsApp, Facebook, YouTube

This site is hosted for free using **GitHub Pages**.

1. Edit the `index.html` file
2. Upload it to this repository (replacing the existing file)
3. Changes go live within 1–2 minutes automatically
